Workplace Productivity Tools: महत्वपूर्ण बिंदु

परीक्षा के दृष्टिकोण से सबसे महत्वपूर्ण टॉपिक्स की सूची:

इस अध्याय में हमें मुख्य रूप से इन 4 बड़े भागों पर ध्यान देना है:

1. मुख्य टॉपिक्स (Core Topics):
  • MS Word: Formatting, Mail Merge, Page Setup, और Header/Footer।
  • MS Excel: Formulas (SUM, IF, VLOOKUP), Cell Referencing, Pivot Tables, और Charts।
  • MS PowerPoint: Slide Master, Transitions, Animations, और Slide Show Settings।
  • Common Tools: Shortcut Keys, File Extensions, और Cloud Storage (OneDrive/Google Drive)।

🚀 एग्जाम-ओरिएंटेड एक्स्ट्रा फैक्ट्स (जरूर याद करें):
  • File Extensions:
    ➔ MS Word: .docx | MS Excel: .xlsx | MS PowerPoint: .pptx
    (नोट: 2007 से पहले के वर्जन्स में 'x' नहीं होता था, जैसे .doc, .xls)
  • Default File Names:
    ➔ Word: Document1 | Excel: Book1 | PowerPoint: Presentation1
  • Maximum Zoom:
    ➔ MS Word में अधिकतम ज़ूम 500% होता है, जबकि Excel और PowerPoint में 400%।
  • Minimum Zoom: तीनों सॉफ्टवेयर्स में न्यूनतम ज़ूम 10% होता है।
  • Control Keys:
    ➔ Ctrl + N: नई फाइल | Ctrl + S: सेव | F12: Save As।

2. Microsoft Word (Word Processing)

दस्तावेज़ बनाने और उन्हें फॉर्मेट करने का सबसे लोकप्रिय टूल:

A. इंटरफ़ेस (Interface):
  • Title Bar: स्क्रीन के सबसे ऊपर, जहाँ फाइल का नाम (Default: Document1) दिखता है।
  • Ribbon: इसमें सभी टैब्स (Home, Insert, आदि) और कमांड्स होते हैं। इसे Ctrl + F1 से हाइड/शो कर सकते हैं।
  • Quick Access Toolbar: बार-बार उपयोग होने वाली कमांड्स जैसे Save, Undo, Redo के लिए।
  • Status Bar: सबसे नीचे, जो पेज नंबर, शब्द संख्या (Word Count) और भाषा दिखाता है।
  • Zoom Slider: स्टेटस बार के दाईं ओर। (Minimum: 10%, Maximum: 500%)।

B. फाइल मेनू (File Menu / Backstage View):
  • New: नया ब्लैंक डॉक्यूमेंट या टेम्पलेट खोलना।
  • Save vs Save As: पहली बार सेव के लिए Save, जबकि फाइल का नाम या लोकेशन बदलने के लिए Save As (F12) का उपयोग होता है।
  • Print: हार्ड कॉपी निकालने के लिए। (Ctrl + P)
  • Export: फाइल को .pdf या .xps फॉर्मेट में बदलने के लिए।

C. होम टैब (Home Tab - सबसे ज्यादा इस्तेमाल होने वाला):

इसमें मुख्य रूप से एडिटिंग और फॉर्मेटिंग टूल्स होते हैं:

1. क्लिपबोर्ड (Clipboard):
  • Cut/Copy/Paste: डेटा को मूव या डुप्लिकेट करना।
  • Format Painter: एक टेक्स्ट की फॉर्मेटिंग (जैसे कलर, फॉन्ट) को दूसरे पर कॉपी करना। (Ctrl + Shift + C)
2. फॉन्ट (Font Group):
  • Change Case: इसमें 5 प्रकार होते हैं: Sentence, lowercase, UPPERCASE, Capitalize Each Word, tOGGLE cASE।
  • Subscript: अक्षर के नीचे छोटा लिखना (जैसे $H_2O$)। (Ctrl + =)
  • Superscript: अक्षर के ऊपर छोटा लिखना (जैसे $A^2$)। (Ctrl + Shift + +)
  • Clear Formatting: सभी स्टाइल हटाकर टेक्स्ट को नॉर्मल बनाना।
3. पैराग्राफ (Paragraph Group):
  • Alignment: Left, Center, Right और Justify (दोनों तरफ से बराबर)।
  • Indentation:
    • First Line Indent: केवल पहली लाइन को आगे खिसकाना।
    • Hanging Indent: पहली लाइन को छोड़कर बाकी पैराग्राफ को खिसकाना।
  • Line Spacing: दो लाइनों के बीच की दूरी। (Default: 1.15)।
📌 महत्वपूर्ण तथ्य:
  • MS Word में डिफॉल्ट फॉन्ट Calibri और साइज 11pt होता है।
  • किसी शब्द को सेलेक्ट करने के लिए उस पर Double Click करें।
  • पूरे पैराग्राफ को सेलेक्ट करने के लिए Triple Click करें।
  • Gutter Margin: वह जगह जो बाइंडिंग (Binding) के लिए छोड़ी जाती है।
  • Ctrl + Enter: नया पेज शुरू करने (Page Break) की शॉर्टकट की है।

3. Insert और Layout टैब (Advanced Formatting)

डॉक्यूमेंट में बाहरी तत्व जोड़ने और पेज की बनावट तय करने के लिए:

A. इंसर्ट टैब (Insert Tab):

इसका उपयोग डॉक्यूमेंट में नई चीज़ें जैसे टेबल, इमेज या हेडर जोड़ने के लिए होता है।

  • Pages:
    ➔ Cover Page: पहले से बने डिज़ाइनर पेज जोड़ना।
    ➔ Page Break (Ctrl + Enter): वर्तमान पेज को खत्म कर अगले पेज पर जाना।
  • Tables:
    ➔ इसमें अधिकतम 63 कॉलम हो सकते हैं।
    ➔ Convert Text to Table: लिखे हुए टेक्स्ट को सीधे टेबल में बदलना।
    ➔ Formulas: टेबल में Layout > Formula का उपयोग करके SUM या AVERAGE जैसे बेसिक हिसाब किए जा सकते हैं।
  • Illustrations:
    ➔ इसमें Shapes, SmartArt (फ्लोचार्ट के लिए) और सीधे Screenshot लेने का विकल्प मिलता है।
  • Header, Footer & Page Number:
    ➔ Header: पेज के सबसे ऊपरी हिस्से में जानकारी (जैसे किताब का नाम)।
    ➔ Footer: पेज के सबसे निचले हिस्से में जानकारी।
  • Symbols:
    ➔ इसमें गणितीय Equations और ऐसे Symbols मिलते हैं जो कीबोर्ड पर नहीं होते (जैसे ©, ®, µ)।

B. लेआउट टैब (Layout Tab):

यह पूरे पेज की दिखावट (Printing Style) को कंट्रोल करता है।

  • Page Setup:
    • Margins: पेज के किनारों पर खाली छोड़ी गई जगह (Normal, Narrow, आदि)।
    • Orientation: पेज खड़ा (Portrait) होगा या लेटा हुआ (Landscape)। डिफॉल्ट Portrait होता है।
    • Size: पेपर का आकार (A4, Letter, Legal)।
    • Columns: टेक्स्ट को अखबार की तरह 2 या 3 कॉलम में बाँटना।
  • Breaks:
    ➔ Page Break: टेक्स्ट को अगले पेज पर भेजता है।
    ➔ Section Break: डॉक्यूमेंट के अलग-अलग हिस्सों में अलग-अलग फॉर्मेटिंग (जैसे अलग पेज नंबर) के लिए।
  • Hyphenation: जब शब्द लाइन के अंत में पूरा नहीं आता, तो यह एक छोटा डैश (-) लगाकर शब्द को अगली लाइन से जोड़ देता है।
📌 महत्वपूर्ण तथ्य:
  • Orientation: MS Word में डिफॉल्ट पेज ओरिएंटेशन Portrait होता है।
  • Margin: टॉप, बॉटम, लेफ्ट और राइट के अलावा बाइंडिंग के लिए छोड़ी गई जगह को Gutter Margin कहते हैं।
  • Section Break: यदि आप एक ही फाइल में कुछ पेज Portrait और कुछ Landscape चाहते हैं, तो Section Break अनिवार्य है।
  • Watermark: पेज के बैकग्राउंड में दिखने वाला हल्का टेक्स्ट (जैसे 'CONFIDENTIAL') Design टैब में मिलता है।
  • Page Number: इसे Header या Footer कहीं भी लगाया जा सकता है।

4. एडवांस टैब्स और फीचर्स (References to View)

डॉक्यूमेंट ऑटोमेशन, प्रूफिंग और मेलिंग के लिए उपयोग होने वाले टूल्स:

A. रेफरेंस टैब (References Tab):

यह बड़ी फाइलों, जैसे थीसिस या किताबों के प्रबंधन के लिए है:

  • Table of Contents: हेडिंग्स के आधार पर ऑटोमेटिक 'विषय-सूची' तैयार करना।
  • Footnote vs Endnote:
    ➔ Footnote: पेज के सबसे नीचे दी गई टिप्पणी।
    ➔ Endnote: पूरे डॉक्यूमेंट या अध्याय के अंत में दी गई टिप्पणी।
  • Citations & Bibliography: किसी लेखक के काम का संदर्भ (Reference) देना।
  • Captions: इमेज या टेबल के नीचे नाम (जैसे Figure 1) जोड़ना।

B. मेलिंग्स टैब (Mailings Tab - अति महत्वपूर्ण):

एक ही लेटर को कई लोगों को एक साथ भेजने की प्रक्रिया:

  • Envelopes & Labels: लिफाफों और लेबल को प्रिंट करना।
  • Mail Merge (मेल मर्ज): इसके मुख्य स्टेप्स हैं:
    1. Select Recipients: उन लोगों की लिस्ट चुनना जिन्हें लेटर भेजना है।
    2. Insert Merge Field: लेटर में नाम, पता आदि के लिए 'Field' सेट करना।
    3. Finish & Merge: फाइनल लेटर तैयार करना।

C. रिव्यु टैब (Review Tab):

डॉक्यूमेंट की गलतियाँ सुधारने और एडिटिंग ट्रैक करने के लिए:

  • Spelling & Grammar: गलतियों को जाँचने के लिए। (F7)
  • Thesaurus: किसी शब्द के समानार्थी (Synonyms) शब्द खोजना। (Shift + F7)
  • Word Count: पेज, शब्द, और कैरेक्टर्स की संख्या देखना।
  • Track Changes: डॉक्यूमेंट में किए गए बदलावों को रिकॉर्ड करना ताकि बाद में उन्हें Accept या Reject किया जा सके।

D. व्यू टैब (View Tab):

यह तय करता है कि डॉक्यूमेंट स्क्रीन पर कैसा दिखेगा:

  • Views:
    ➔ Read Mode: पढ़ने के लिए बेस्ट।
    ➔ Print Layout: डिफॉल्ट व्यू (जैसा प्रिंट होगा वैसा दिखेगा)।
    ➔ Web Layout: वेबसाइट पेज की तरह।
    ➔ Outline/Draft: केवल टेक्स्ट और हेडिंग्स पर फोकस करने के लिए।
  • Navigation Pane: डॉक्यूमेंट में किसी विशेष हेडिंग या पेज पर जल्दी जाने के लिए।
  • Macros: बार-बार होने वाले कार्यों को 'रिकॉर्ड' करके ऑटोमेटिक करना।
📌 महत्वपूर्ण तथ्य:
  • Thesaurus: क्या है? ➔ उत्तर: समानार्थी शब्दों का शब्दकोश।
  • Mail Merge: का मुख्य उद्देश्य क्या है? ➔ उत्तर: एक ही सूचना को कम समय में कई लोगों तक पहुँचाना।
  • Macros: किस भाषा (Script) का उपयोग करते हैं? ➔ उत्तर: VBA (Visual Basic for Applications)।
  • Default View: MS Word का डिफॉल्ट व्यू Print Layout होता है।
  • F7: स्पेलिंग चेक के लिए | Shift + F7: थिसॉरस (समानार्थी) के लिए।

वर्कप्लेस टूल्स का महत्व:

आजकल बैंकिंग और SSC जैसे एग्जाम्स में "Working with MS Office" से प्रैक्टिकल सवाल (जैसे: "अक्षर को सुपरस्क्रिप्ट करने की शॉर्टकट की क्या है?") अधिक पूछे जा रहे हैं।

5. Microsoft Excel (Spreadsheets)

डेटा को रो (Rows) और कॉलम (Columns) में व्यवस्थित करने और गणना करने का टूल:

A. एक्सेल के बुनियादी तत्व (Fundamentals):
  • Workbook vs Worksheet: एक्सेल की एक फाइल Workbook कहलाती है, जबकि इसके अंदर के पेजों को Worksheet कहते हैं। एक वर्कबुक में कई शीट्स हो सकती हैं।
  • रो और कॉलम (Rows & Columns):
    ➔ Rows: ये क्षैतिज (Horizontal) होती हैं और इन्हें नंबर (1, 2, 3...) से दर्शाया जाता है। कुल संख्या: 1,048,576।
    ➔ Columns: ये लंबवत (Vertical) होते हैं और इन्हें अक्षरों (A, B, C...) से दर्शाया जाता है। कुल संख्या: 16,384 (अंतिम कॉलम का नाम: XFD)।
  • Cell Address & Name Box: रो और कॉलम के मिलन स्थान को Cell कहते हैं।
    ➔ Name Box: एक्टिव सेल का एड्रेस (जैसे A1) दिखाता है।
    ➔ Formula Bar: सेल के अंदर लिखा गया डेटा या फॉर्मूला यहाँ दिखाई देता है।

B. डेटा एंट्री टूल्स (Data Entry):

एक्सेल में काम को तेज़ बनाने के लिए इन फीचर्स का उपयोग होता है:

  • AutoFill: किसी पैटर्न (जैसे 1, 2, 3 या Sunday, Monday) को सेल के कोने से खींचकर (Drag) अपने आप भरना।
  • Custom Lists: अपनी स्वयं की लिस्ट बनाना ताकि बार-बार टाइप न करना पड़े।
  • Flash Fill (Ctrl + E): यह डेटा के पैटर्न को समझकर बाकी सेल्स को अपने आप भर देता है (जैसे नाम और सरनेम अलग करना)।

C. फॉर्मेटिंग (Formatting):
  • Merge & Center: कई सेल्स को मिलाकर एक बड़ा सेल बनाना और टेक्स्ट को बीच में लाना।
  • Wrap Text: यदि टेक्स्ट सेल से बाहर जा रहा है, तो उसे उसी सेल में कई लाइनों में फिट करना।
  • Conditional Formatting: किसी शर्त के आधार पर सेल का रंग या स्टाइल बदलना (जैसे: 33 से कम नंबर वालों को लाल रंग में दिखाना)।
  • Number Formats: डेटा को Currency (₹), Accounting, Percentage (%) या Date/Time के रूप में सेट करना।
📌 महत्वपूर्ण तथ्य:
  • एक्सेल की एक वर्कशीट में कुल 17,179,869,184 सेल्स होते हैं।
  • किसी भी सेल की चौड़ाई को ऑटो-फिट करने के लिए कॉलम हेडर पर Double Click करें।
  • Cell Address: हमेशा पहले कॉलम लेटर और फिर रो नंबर आता है (जैसे B5)।
  • Sheet Tab: वर्कशीट के नाम बदलने या नए शीट जोड़ने के लिए सबसे नीचे स्थित होता है।
  • Ctrl + Page Up/Down: एक शीट से दूसरी शीट पर जाने की शॉर्टकट की है।

6. Excel Formulas और Data Management

गणना और डेटा विश्लेषण के लिए सबसे शक्तिशाली टूल्स:

A. फॉर्मूला और फंक्शन्स (Formulas & Functions):

एक्सेल में हर फॉर्मूला बराबर (=) के चिह्न से शुरू होता है।

1. गणितीय (Math Functions):
  • SUM: रेंज की सभी संख्याओं को जोड़ना। उदा: `=SUM(A1:A5)`
  • PRODUCT: संख्याओं का गुणा करना। उदा: `=PRODUCT(5,2)` ➔ 10
  • MOD: भाग देने के बाद शेषफल (Remainder) निकालना। उदा: `=MOD(10,3)` ➔ 1
  • ROUND: दशमलव वाली संख्या को पूर्णांक में बदलना। उदा: `=ROUND(15.67, 1)` ➔ 15.7
  • SUBTOTAL: केवल फिल्टर किए गए डेटा का जोड़ या औसत निकालना।
2. टेक्स्ट (Text Functions):
  • CONCATENATE / TEXTJOIN: दो या अधिक सेल्स के टेक्स्ट को जोड़ना।
  • LEFT/RIGHT/MID: टेक्स्ट के बाएं, दाएं या बीच से कुछ अक्षर अलग करना।
  • LEN: सेल में मौजूद कुल अक्षरों की संख्या बताना। उदा: `=LEN("Asha")` ➔ 4
  • TRIM: टेक्स्ट के आगे और पीछे के फालतू 'Spaces' हटाना।
  • UPPER/LOWER: टेक्स्ट को बड़े या छोटे अक्षरों में बदलना।
3. लॉजिकल (Logical Functions):
  • IF: शर्त के आधार पर परिणाम देना। उदा: `=IF(A1>=33, "Pass", "Fail")`
  • AND/OR: एक से अधिक शर्तों को चेक करना।
  • IFERROR: यदि फॉर्मूले में कोई एरर (जैसे #DIV/0!) आए, तो उसकी जगह कोई मैसेज दिखाना।
4. लुकअप (Lookup Functions):
  • VLOOKUP: डेटा को Vertical (खड़े रूप में) खोजना। (सबसे ज्यादा पूछा जाने वाला)
  • HLOOKUP: डेटा को Horizontal (आड़े रूप में) खोजना।
  • Match & Index: इनका मेल VLOOKUP से भी ज्यादा फ्लेक्सिबल होता है और डेटा को किसी भी दिशा में खोज सकता है।

B. डेटा टैब (Data Tab):

यह टैब बड़े डेटासेट को मैनेज करने के लिए उपयोग होता है:

  • Sort: डेटा को A-Z या Z-A क्रम में व्यवस्थित करना (Single या Multi-level)।
  • Filter: केवल वही डेटा देखना जो आपकी शर्त को पूरा करे।
  • Data Validation: यह तय करना कि सेल में क्या टाइप किया जा सकता है (जैसे केवल 1 से 100 के बीच की संख्या)।
  • Remove Duplicates: एक जैसे दिखने वाले डेटा (Duplicate entries) को डिलीट करना।
  • Text to Columns: एक सेल में लिखे डेटा को अलग-अलग कॉलम में बाँटना (जैसे 'Full Name' से 'First' और 'Last Name' अलग करना)।
📌 महत्वपूर्ण तथ्य:
  • Relative Reference: सेल कॉपी करने पर एड्रेस बदल जाता है (उदा: A1)।
  • Absolute Reference: सेल एड्रेस को लॉक करने के लिए $ का उपयोग होता है (उदा: $A$1)। इसकी शॉर्टकट की F4 है।
  • Circular Reference: जब कोई फॉर्मूला अपने ही सेल को रेफर करता है।
  • #VALUE!: यह एरर तब आता है जब फॉर्मूले में गलत डेटा टाइप (जैसे नंबर की जगह टेक्स्ट) इस्तेमाल हो।
  • Ctrl + Shift + L: फिल्टर (Filter) लगाने या हटाने की शॉर्टकट की है।

7. डेटा विज़ुअलाइज़ेशन और एडवांस एनालिसिस

डेटा को ग्राफ़ में बदलने और उसे सुरक्षित रखने के तरीके:

A. चार्ट्स (Charts):

डेटा को चित्रों (Graphics) के माध्यम से समझने के लिए इनका उपयोग होता है:

  • Column Chart: समय के साथ डेटा में बदलाव दिखाने के लिए (Vertical Bars)।
  • Bar Chart: अलग-अलग श्रेणियों (Categories) के बीच तुलना के लिए (Horizontal Bars)।
  • Line Chart: ट्रेंड (Trend) दिखाने के लिए, जैसे शेयर मार्केट या तापमान।
  • Pie Chart: एक पूरी चीज़ के अलग-अलग हिस्सों (Proportions) को दिखाने के लिए।
  • Scatter Chart: दो वेरिएबल्स के बीच संबंध (Correlation) दिखाने के लिए।
  • Pivot Chart: पिवट टेबल के डेटा को सीधे चार्ट में बदलना।
[Image of different types of Excel charts Column Bar Line and Pie]
B. डेटा एनालिसिस (Analysis Tools):
  • Pivot Table (पिवट टेबल): यह बड़े और जटिल डेटा का **Summary (सारांश)** तैयार करने का सबसे तेज़ तरीका है। इससे आप बिना किसी फॉर्मूले के डेटा को ग्रुप या फ़िल्टर कर सकते हैं।
  • What-If Analysis:
    • Goal Seek: जब आपको पता हो कि 'आउटपुट' क्या चाहिए, लेकिन यह नहीं पता कि उस तक पहुँचने के लिए 'इनपुट' में क्या बदलाव करना होगा।
    • Scenario Manager: अलग-अलग स्थितियों (Scenarios) के लिए अलग-अलग वैल्यू सेट करके परिणाम देखना।

C. सुरक्षा (Security):

डेटा को अनधिकृत बदलावों से बचाने के लिए:

  • Protect Sheet: इसके द्वारा आप सेल में बदलाव करने, रो/कॉलम डिलीट करने या फॉर्मेटिंग करने पर रोक लगा सकते हैं।
  • Protect Workbook: यह वर्कबुक के स्ट्रक्चर को सुरक्षित करता है, ताकि कोई नई शीट जोड़ या डिलीट न कर सके।
  • Lock Cells: डिफॉल्ट रूप से सभी सेल्स 'Locked' होते हैं, लेकिन यह तभी प्रभावी होता है जब आप शीट को 'Protect' करते हैं।
📌 महत्वपूर्ण तथ्य:
  • F11: सेलेक्ट किए गए डेटा से तुरंत Chart बनाने की शॉर्टकट की है।
  • Alt + F1: उसी शीट पर चार्ट (Embedded Chart) इंसर्ट करने के लिए।
  • Legend: चार्ट में दिखने वाले वो कलर बॉक्स जो बताते हैं कि कौन सा रंग किस डेटा को दर्शा रहा है।
  • Pivot Table: का उपयोग डेटा को 'Summarize' करने के लिए होता है।
  • Goal Seek: केवल एक वेरिएबल (Variable) के साथ काम कर सकता है।

8. Microsoft PowerPoint (Presentation)

प्रभावशाली प्रेजेंटेशन और स्लाइड्स बनाने का मुख्य टूल:

A. स्लाइड्स (Slides) और उनकी सेटिंग्स:

पावरपॉइंट का एक पेज 'स्लाइड' कहलाता है।

  • New Slide (Ctrl + M): प्रेजेंटेशन में एक नई खाली स्लाइड जोड़ने के लिए।
  • Duplicate Slide (Ctrl + D): चुनी गई स्लाइड की हूबहू नकल तैयार करने के लिए।
  • Slide Layouts: इसमें कुल 9 प्रकार के लेआउट होते हैं (जैसे: Title Slide, Title and Content, Two Content, Comparison, Blank आदि)।
  • Reset: स्लाइड की पोजीशन और फॉर्मेटिंग को वापस डिफॉल्ट पर सेट करना।

B. व्यूज़ (Views - स्लाइड्स को देखने के तरीके):
  • Normal View: यह डिफॉल्ट व्यू है जहाँ आप स्लाइड को एडिट और डिज़ाइन करते हैं।
  • Slide Sorter View: इसमें सभी स्लाइड्स छोटी-छोटी 'Thumbnails' के रूप में दिखती हैं। यह स्लाइड्स का क्रम (Order) बदलने के लिए सबसे अच्छा है।
  • Notes Page: इसमें स्लाइड के नीचे स्पीकर के लिए 'Notes' लिखने की जगह होती है।
  • Reading View: बिना फुल स्क्रीन किए प्रेजेंटेशन को पढ़ने के लिए।
  • Slide Master: यह सबसे शक्तिशाली टूल है। यहाँ किया गया बदलाव (जैसे लोगो लगाना या फॉन्ट बदलना) सभी स्लाइड्स पर अपने आप लागू हो जाता है।

C. ट्रांज़िशन (Transitions):

जब एक स्लाइड खत्म होकर दूसरी स्लाइड आती है, तो उसके बीच के इफेक्ट को 'Transition' कहते हैं।

  • Duration: इफेक्ट कितनी देर तक चलेगा (सेकंड में)।
  • Sound: स्लाइड बदलते समय कोई आवाज़ जोड़ना।
  • Advance Slide: स्लाइड माउस क्लिक पर बदलेगी या एक निश्चित समय (After) के बाद।

D. एनिमेशन (Animations):

स्लाइड के अंदर मौजूद ऑब्जेक्ट्स (Text, Image, Shape) पर लगने वाले इफेक्ट्स। इसके 4 मुख्य प्रकार हैं:

  • Entrance (प्रवेश): ऑब्जेक्ट स्क्रीन पर कैसे आएगा (जैसे: Fly In, Fade)।
  • Emphasis (ज़ोर देना): ऑब्जेक्ट स्क्रीन पर रहते हुए कैसे रिएक्ट करेगा (जैसे: Spin, Grow)।
  • Exit (बाहर जाना): ऑब्जेक्ट स्क्रीन से कैसे गायब होगा।
  • Motion Paths: ऑब्जेक्ट को एक निश्चित दिशा या लाइन में चलाना।

E. स्लाइड शो (Slide Show):
  • From Beginning (F5): प्रेजेंटेशन को पहली स्लाइड से शुरू करना।
  • From Current Slide (Shift + F5): जिस स्लाइड पर आप अभी हैं, वहीं से शो शुरू करना।
  • Presenter View: इसमें प्रेजेंटर को अपनी स्क्रीन पर नोट्स और अगली स्लाइड दिखती है, जबकि ऑडियंस को केवल करंट स्लाइड।
  • Rehearse Timings: प्रेजेंटेशन की प्रैक्टिस करना ताकि यह पता चल सके कि हर स्लाइड पर कितना समय लग रहा है।
📌 महत्वपूर्ण तथ्य:
  • पावरपॉइंट फाइल का डिफॉल्ट नाम Presentation1 होता है।
  • स्लाइड शो को बीच में रोकने के लिए Esc की (Key) का उपयोग होता है।
  • Handouts: प्रेजेंटेशन की स्लाइड्स को कागज़ पर प्रिंट करना (एक पेज पर 1 से 9 स्लाइड्स तक)।
  • Extension: .pptx (नया वर्ज़न) और .ppsx (केवल स्लाइड शो फाइल)।
  • Ctrl + K: स्लाइड में हाइपरलिंक (Hyperlink) जोड़ने के लिए।

9. Google Workspace (Cloud-based Tools)

इंटरनेट और क्लाउड पर आधारित कार्यक्षमता की समझ:

A. Google Docs, Sheets और Slides:

ये Microsoft Office के ऑनलाइन विकल्प हैं। इनके मुख्य अंतर और विशेषताएं नीचे दी गई हैं:

  • Auto-save: क्लाउड टूल्स में आपको बार-बार सेव करने की ज़रूरत नहीं होती। हर बदलाव अपने आप तुरंत सेव हो जाता है।
  • Real-time Collaboration: एक ही समय पर कई लोग (Team) एक ही फाइल पर काम कर सकते हैं और एक-दूसरे के बदलाव देख सकते हैं।
  • Version History: आप फाइल के पुराने वर्जन्स को कभी भी देख सकते हैं और उन्हें रिस्टोर (Restore) कर सकते हैं।
  • Offline Mode: बिना इंटरनेट के भी काम किया जा सकता है, जो इंटरनेट आने पर अपने आप सिंक (Sync) हो जाता है।

B. Google Drive (क्लाउड स्टोरेज):

यह एक डिजिटल लॉकर की तरह है जहाँ आपकी सभी फाइलें सुरक्षित रहती हैं।

  • Uploading/Downloading: कंप्यूटर की फाइलों को क्लाउड पर डालना (Upload) और क्लाउड से कंप्यूटर में लेना (Download)।
  • Search Operators: ड्राइव में फाइल खोजने के लिए `type:pdf` या `owner:me` जैसे ऑपरेटर्स का उपयोग होता है।
  • 'Shared with me': इस सेक्शन में वे फाइलें दिखती हैं जिन्हें किसी और ने आपके साथ शेयर किया है।
  • Storage: व्यक्तिगत Google अकाउंट के साथ 15 GB मुफ्त स्टोरेज मिलता है।

C. शेयरिंग परमिशन (Sharing Permissions):

जब आप फाइल शेयर करते हैं, तो आप यह कंट्रोल कर सकते हैं कि सामने वाला क्या कर सकता है:

  • Viewer: केवल फाइल को देख सकता है, कोई बदलाव नहीं कर सकता।
  • Commenter: बदलाव नहीं कर सकता, लेकिन सुझाव (Comments) दे सकता है।
  • Editor: फाइल में कुछ भी लिख सकता है, डिलीट कर सकता है या दूसरों के साथ शेयर कर सकता है।

D. Google Forms (सर्वे और डेटा कलेक्शन):

इसका उपयोग ऑनलाइन टेस्ट या डेटा इकठ्ठा करने के लिए किया जाता है।

  • Question Types: इसमें कई प्रकार के प्रश्न होते हैं:
    • Multiple Choice: केवल एक सही विकल्प चुनने के लिए।
    • Checkboxes: एक से अधिक विकल्प चुनने के लिए।
    • Dropdown: लिस्ट में से एक विकल्प चुनने के लिए।
    • Short Answer/Paragraph: टेक्स्ट लिखने के लिए।
  • Responses: प्राप्त हुए जवाबों को सीधे Google Sheets में एक्सपोर्ट (Export) करके उनका विश्लेषण किया जा सकता है।
📌 महत्वपूर्ण तथ्य:
  • Google Docs का एक्सटेंशन (फाइल फॉर्मेट) वेब पर नहीं होता, लेकिन डाउनलोड करने पर यह .docx में बदल जाता है।
  • Cloud Computing: इंटरनेट के माध्यम से कंप्यूटर संसाधनों (जैसे स्टोरेज, सॉफ्टवेयर) का उपयोग करना 'क्लाउड कंप्यूटिंग' कहलाता है।
  • URL Shortener: Google Forms के लंबे लिंक को छोटा करने का विकल्प 'Send' बटन के अंदर मिलता है।
  • Google Keep: इसका उपयोग जल्दी से 'Notes' लेने के लिए किया जाता है।

10. ईमेल और डिजिटल कम्युनिकेशन

इंटरनेट पर संदेश भेजने और प्राप्त करने की तकनीकी प्रक्रिया:

A. ईमेल की संरचना (Structure of an Email):

एक प्रोफेशनल ईमेल मुख्य रूप से निम्नलिखित भागों में बँटा होता है:

  • Header (हेडर):
    • To: मुख्य प्राप्तकर्ता का ईमेल एड्रेस।
    • CC (Carbon Copy): उन लोगों के लिए जिन्हें जानकारी में रखना है (सबको एक-दूसरे के एड्रेस दिखते हैं)।
    • BCC (Blind Carbon Copy): इसमें जो एड्रेस होगा, वह अन्य प्राप्तकर्ताओं को दिखाई नहीं देगा।
    • Subject: ईमेल का संक्षिप्त विषय (प्रोफेशनल ईमेल के लिए अनिवार्य)।
  • Body: ईमेल का मुख्य संदेश।
  • Attachments: फाइलें (Documents, Images) जो ईमेल के साथ भेजी जाती हैं। (Gmail में सीमा: 25 MB)
  • Signature: ईमेल के अंत में भेजने वाले का नाम, पद और संपर्क जानकारी जो अपने आप जुड़ जाती है।

B. ईमेल प्रोटोकॉल्स (Email Protocols - Technical):

ईमेल भेजने और प्राप्त करने के लिए कुछ नियमों (Protocols) का पालन किया जाता है:

Protocol Full Form मुख्य कार्य Port Number
SMTP Simple Mail Transfer Protocol ईमेल को भेजने (Send) के लिए। 25, 587
POP3 Post Office Protocol v3 सर्वर से ईमेल डाउनलोड करने के लिए (एक डिवाइस के लिए)। 110, 995 (SSL)
IMAP Internet Message Access Protocol ईमेल को सर्वर पर ही मैनेज करने के लिए (मल्टीपल डिवाइस सिंक)। 143, 993 (SSL)

C. ईमेल मैनेजमेंट (Management):
  • Folders:
    ➔ Inbox: प्राप्त हुए ईमेल।
    ➔ Drafts: वे ईमेल जो लिखे गए हैं पर भेजे नहीं गए।
    ➔ Spam: अवांछित (Unsolicited) या जंक ईमेल।
    ➔ Trash/Bin: डिलीट किए गए ईमेल (अक्सर 30 दिनों तक यहाँ रहते हैं)।
  • Archive vs Delete:
    ➔ Archive: ईमेल को इनबॉक्स से हटाकर सुरक्षित रखना (डिलीट नहीं होता)।
    ➔ Delete: ईमेल को पूरी तरह हटा देना।
  • Filters/Rules: आने वाले ईमेल को अपने आप किसी फोल्डर में डालना या लेबल करना।
📌 महत्वपूर्ण तथ्य:
  • पहला ईमेल 1971 में रे टॉमलिंसन (Ray Tomlinson) ने भेजा था।
  • ईमेल एड्रेस के दो भाग होते हैं: Username और Domain Name, जो @ चिह्न से जुड़े होते हैं।
  • Spam: को तकनीकी भाषा में UCE (Unsolicited Commercial Email) भी कहा जाता है।
  • Bounce Back: जब गलत एड्रेस के कारण ईमेल वापस आ जाता है।
  • Phishing: फर्जी ईमेल भेजकर यूजर की निजी जानकारी चुराना।

11. कॉमन शॉर्टकट्स, एक्सटेंशन और प्रिंटिंग

परीक्षा में समय बचाने और तकनीकी बारीकियों को समझने के लिए अनिवार्य:

A. मास्टर कीबोर्ड शॉर्टकट्स (Ctrl + A to Z):

ये शॉर्टकट्स लगभग सभी MS Office टूल्स में समान रूप से काम करते हैं:

Ctrl + A: Select All Ctrl + B: Bold Ctrl + C: Copy
Ctrl + D: Font Window (Word) / Duplicate (PPT) Ctrl + E: Center Alignment Ctrl + F: Find
Ctrl + G: Go To Ctrl + H: Replace Ctrl + I: Italic
Ctrl + J: Justify Alignment Ctrl + K: Insert Hyperlink Ctrl + L: Left Alignment
Ctrl + M: Indent (Word) / New Slide (PPT) Ctrl + N: New File Ctrl + O: Open File
Ctrl + P: Print Ctrl + Q: Reset Paragraph Ctrl + R: Right Alignment
Ctrl + S: Save Ctrl + T: Hanging Indent Ctrl + U: Underline
Ctrl + V: Paste Ctrl + W: Close Document Ctrl + X: Cut
Ctrl + Y: Redo Ctrl + Z: Undo

B. फंक्शन कीज़ (Function Keys):
  • F1: Help (मदद के लिए)।
  • F2: फाइल को Rename करना या Excel में सेल को Edit करना।
  • F5: Refresh करना / Go To खोलना / PowerPoint में Slide Show शुरू करना।
  • F7: Spelling और Grammar चेक करना।
  • F12: Save As विंडो खोलना।

C. फाइल एक्सटेंशन (File Extensions):

एक्सटेंशन यह बताता है कि फाइल किस प्रकार की है:

  • .docx: MS Word डॉक्यूमेंट (2007 और बाद के)।
  • .dotx: MS Word टेम्पलेट फाइल।
  • .xlsx: MS Excel स्प्रेडशीट।
  • .csv: Comma Separated Values (सादा डेटा फाइल)।
  • .pptx: MS PowerPoint प्रेजेंटेशन।
  • .ppsx: PowerPoint स्लाइड शो (सीधे शो खुलता है)।
  • .pdf: Portable Document Format (अपरिवर्तनीय फाइल)।
  • .txt: प्लेन टेक्स्ट (Notepad)।
  • .rtf: Rich Text Format (WordPad)।

D. प्रिंटिंग कॉन्सेप्ट्स (Printing):
  • Page Range:
    ➔ 1-5: पेज 1 से 5 तक लगातार।
    ➔ 8, 11: केवल पेज 8 और पेज 11।
  • Collate (कोलेट): यदि आप 10 पेज के डॉक्यूमेंट की 3 कॉपी निकाल रहे हैं, तो Collate ऑन होने पर पहले 1-10 का एक सेट निकलेगा, फिर दूसरा, फिर तीसरा।
  • Orientation: Portrait (खड़ा) या Landscape (पड़ा)।
  • Scale to Fit: बड़े डेटा को एक ही पेज पर सिकोड़ कर फिट करना (विशेषकर Excel में)।
📌 परीक्षा के लिए रामबाण तथ्य:
  • Ctrl + [ / ]: फॉन्ट साइज को घटाने या बढ़ाने के लिए।
  • Shift + F3: टेक्स्ट का 'Case' बदलने के लिए (Change Case)।
  • Alt + Enter: Excel में एक ही सेल के अंदर नई लाइन शुरू करने के लिए।
  • Ctrl + Semicolon (;): Excel में आज की तारीख (Current Date) डालने के लिए।
  • Print Preview: प्रिंट करने से पहले यह देखना कि पेज कैसा दिखेगा। (Ctrl + F2)
